F5 Products and Concepts

TMM
Traffic Management
TMOS
Big IP Traffic Management Operating System, the functionality can be considered modular, module entries come next

Modules

LTM
Local Traffic Management
Virtual servers, Pools, Profiles, Policies, Full Proxy, SSL manipulation
DNS/GTM
Global Traffic Management
this has functionality similar to Cloudflare, includes a intelligent and saclable DNS Solution, DNS Sec, DR capability to swap DC, direct traffic to best performing DC
APM
Access Policy Management
Access control, identify federation, SSO, SAML/OAuth, VPN (layer 3)
SWG
Secure Web Gateway
Forward proxy for outbound traffic, Filter, Block social media
ASM
Application Security Manager
Layer 7 Web Application Firewall - WAF, to protect against the OWASP top 10 attacks
AFM
Advanced Firewall Manager
Helps against Layer 3 / Layer 4 attacks, bandwith management, latency reduction

Other products

Big·IQ
Management solution to manage Big·IP products
Silverline
Cloud based service to protect cloud services from DoS atacks
F5SaaS
F5 as a service in the cloud
